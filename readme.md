### *Multi-Category Vacation Photo Organizer*

[![Android](https://img.shields.io/badge/Android-3DDC84?style=flat&logo=android&logoColor=white)](https://developer.android.com/) [![Java](https://img.shields.io/badge/Java-007396?style=flat&logo=java&logoColor=white)](https://www.java.com/) [![SQLite](https://img.shields.io/badge/SQLite-003B57?style=flat&logo=sqlite&logoColor=white)](https://www.sqlite.org/) [![XML](https://img.shields.io/badge/XML-FFA500?style=flat&logo=xml&logoColor=white)](https://www.w3.org/XML/)

**Travelog** is a native Android application designed to solve the chaos of unsorted vacation photos. Built with **Java** and **SQLite**, it empowers users to organize memories into custom, flexible categories. Unlike standard galleries that sort only by date, Travelog allows for thematic organization‚Äîletting you group photos by specific events, locations, or experiences.

---

## üöÄ Key Features

* **Custom Categorization:** Create specific albums for parts of your trip (e.g., *"Car Museum"*, *"Archery Lessons"*, *"Day 1 in LA"*).
* **Multi-Tagging System:** A single photo can belong to multiple categories simultaneously. A group selfie can live in both *"Family"* and *"Beach Day"* without duplicating the file.
* **Offline First:** Utilizing **SQLite** for local storage, the app is fully functional without an internet connection‚Äîperfect for travel in remote areas.
* **High-Contrast UI:** Designed with accessibility in mind, featuring clear button labels and high-contrast visuals for easy navigation in bright sunlight.
